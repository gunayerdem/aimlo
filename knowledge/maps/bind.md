# BIND — Detaylı Strateji ve Analiz

## Harita Karakteristikleri
- 2 site, MID YOK (unique)
- Teleporter sistemi (A Short → B Short, B Short → A Short)
- Hızlı rotasyon imkanı
- Site'lar geniş ve açık
- Teleporter sesi = bilgi

## SALDIRI STRATEJİLERİ

### Default
- 2 kişi A Short/Bath (bilgi)
- 3 kişi B Long/Short (kontrolü)
- Teleporter ile hızlı fake/rotate

### A Execute
- Smoke: Heaven + A Lamps
- Flash: A Bath'tan site'a
- Teleporter ile B'den 1 kişi A Short'a flank
- Plant: A Default (Triple Box arkası)
- Post-plant: Lamps + A Short crossfire

### B Execute
- Smoke: B Elbow + B Window (Hookah)
- B Long push: 3 kişi + utility dump
- Hookah kontrolü kritik (üstten flank)
- Plant: B Default
- Post-plant: B Garden + B Long crossfire

### Teleporter Strats
- Fake A (utility) → Teleporter → B rush
- B'den 2 kişi Teleporter'a → A'ya split
- Teleporter sesi aldatıcı: koyup girmemek = bilgi kontrolü

## SAVUNMA STRATEJİLERİ

### A Site
- Heaven = güçlü ama Op'a karşı riskli
- Lamps arkası off-angle
- Triple Box'ta anchor
- A Short'ta erken bilgi → rotate kararı

### B Site
- B Elbow = çok güçlü anchor pozisyonu
- Hookah kontrolü = B güvenliği
- B Garden'da geri oyna, choke point'i kullan
- B Window/Hookah tripwire = erken uyarı

## YAYGĞIN HATALAR
- Teleporter'a atıp düşmanın sesi duymasını beklemeden push
- A Bath'ta sıkışmak (dar koridor, crossfire yok)
- B Long'da 5-man push'a çekilmemek
- Hookah kontrolünü vermek
- Teleporter sesini ignore etmek

## PRO META
- Bind: Raze + Skye + Brimstone çok güçlü
- Raze boombot B Long = bedava bilgi
- Skye flash + Raze satchel = agresif entry
- Brimstone post-plant molly = guaranteed round
- Viper Bind'da zayıf (wall iyi pozisyon bulamıyor)
