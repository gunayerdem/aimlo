# LOTUS — Detaylı Strateji ve Analiz

## Harita Karakteristikleri
- 3 site (A, B, C) — geniş harita, çok sayıda rotasyon yolu
- Dönebilen kapılar (Rotating Doors): A Main ve C Mound arasında
- Yıkılabilir duvar: B site'ta (kırıldığında yeni giriş açılır)
- Silent Drop: A site'a sessiz düşüş noktası
- Geniş mid bölgesi: birden fazla bağlantı noktası
- Attacker-sided harita — çok fazla giriş noktası var

## SALDIRI STRATEJİLERİ

### Default (1-2-2 veya 2-1-2)
- 1-2 kişi A Main (Root kontrolü, bilgi)
- 1-2 kişi Mid/B (Rubble, Mid Link)
- 2 kişi C Main/Mound (bilgi al)
- Dönebilen kapıları ses olarak kullan — fake rotate sesi
- Yıkılabilir duvarı kır veya kırmayarak bilgi manipüle et

### A Execute
- Smoke: A Heaven (Tree) + A Stairs (CT tarafı)
- Flash: A Main'den site'a
- A Root kontrolü şart — Root'siz A execute flank riski
- Silent Drop kullan: gürültüsüz A site'a 1 kişi gönder
- Entry: A Main → A Site, Silent Drop'tan destek
- Spike: A Default (kutuların arkası)
- Post-plant: A Main + A Root crossfire

### B Execute
- Smoke: B Upper (CT) + B Pillar
- Yıkılabilir duvarı erken kır (Raze, Breach, Judge) veya fake olarak kullan
- B Main → B Site: utility dump zorunlu
- B Upper'dan gelen rotate'u smoke'la
- Entry: B Main push + duvar kırılmışsa ikinci giriş
- Spike: B Default
- Post-plant: B Main + kırık duvar arkası crossfire

### C Execute
- Smoke: C Waterfall + C Mound (CT tarafı)
- Flash: C Main'den Mound'a doğru
- C Mound kontrolü çok önemli (yükseklik avantajı)
- Dönebilen kapı ile A'dan C'ye hızlı rotate (veya tersi)
- Entry: C Main → C Mound → C Site
- Spike: C Default (Waterfall tarafı)
- Post-plant: C Main + C Mound crossfire

### Rotating Door Strategies
- A'dan C'ye (veya tersi) kapıyla hızlı rotate — defender'ı şaşırtır
- Kapı sesini fake olarak kullan: aç ama geçme
- Kapı açıkken push yapan düşmanı bekleyerek tuzak kur
- Eco round'da kapı rotasyonu ile hızlı site switch

## SAVUNMA STRATEJİLERİ

### Default Savunma (2-1-2)
- A: 1 A Tree/Heaven, 1 A Root/Rubble
- B: 1 kişi B Site (anchor + duvar bilgisi)
- C: 1 C Mound/Waterfall, 1 C Main bilgi

### A Site Savunma
- A Tree (Heaven) = güçlü anchor, açı avantajı
- A Root kontrolü ile flank bilgisi
- Silent Drop'a dikkat — sessiz giriş olabilir
- A Stairs'te off-angle, A Main push'ı izle

### B Site Savunma
- Yıkılabilir duvar = B'nin en büyük zayıflığı ve gücü
- Duvar sağlamsa: sadece B Main'i izle
- Duvar kırıldıysa: 2 giriş noktası, crossfire kur
- B Upper rotasyonu hızlı yapılmalı
- KJ/Cypher B'de çok güçlü — utility ile delay

### C Site Savunma
- C Waterfall arkası = güçlü anchor pozisyonu
- C Mound'da agresif oynama — kill al geri çekil
- C Main'de early bilgi al, agresif peek riskli
- C Connector ile B'ye veya Mid'e rotate

### Dönebilen Kapı Savunması
- Kapı sesini dinle — rotate sinyali olabilir
- Kapı açıldığında karşı taraftan peek
- Kapıyı kapatarak rotasyonu yavaşlat

## YAYGIN HATALAR (Rank Bazlı)

### Gold-Plat
- Yıkılabilir duvarı her round 1. saniyede kırmak (bilgi veriyorsun)
- Rotating door'u kullanmamak (haritanın en güçlü mekanikleri)
- 3 site'ı 2-2-1 tutup mid'i boş bırakmak
- Silent Drop'u bilmemek veya kullanmamak
- C Mound'da açıkta durma (sightline çok uzun)

### Diamond-Ascendant
- Over-rotate (3 site = fake'lere düşme riski yüksek)
- Duvar kırılma timing'ini yanlış okuma
- Post-plant'te kapı kullanmayı unutma
- B anchor yardım istemeden ölmesi
- A Tree'de çok uzun durma (flush out utility)

### Immortal+
- Kapı mekaniklerini anti-strat'a dahil etmemek
- Aynı site'a 3 round üst üste execute (okunur)
- Yıkılabilir duvar timing'iyle mind game yapmamak
- Mid-round karar gecikmesi (3 site = hızlı adaptasyon lazım)
- Timeout stratejik kullanmamak

## PRO TAKIMLAR — LOTUS META

### Fnatic Lotus Stili
- Alfajer aggressive B site plays (duvar kırma timing)
- Leo Initiator utility 3 site bilgi toplama
- Split execute: A + C kapı kullanarak
- Boaster timeout masterclass

### LOUD Lotus Stili
- Less Cypher 3 site bilgi ağı
- aspas lurk kapı rotasyonlarıyla
- Cauanzin entry C Main agresif push
- Post-plant lineup çok güçlü (Viper + Brimstone)

### Gen.G Lotus Stili
- Methodical A execute (Root kontrolü + Silent Drop)
- t3xture Op C Long
- Lakia lurk B kırık duvar
- Ekonomi yönetimi çok disiplinli

## AGENT-SPECIFIC İPUÇLARI
- Raze: Yıkılabilir duvarı blast pack ile kır, B Main satchel entry
- Breach: Duvar kırma + B Main fault line, A Main flash combo
- Killjoy: B anchor (turret + alarm duvarı izlesin), lockdown B
- Fade: Haunt 3 site bilgi toplama, Prowler kapı arkasını kontrol
- Omen: TP ile kapı bypass, C Waterfall'a TP
- Viper: Wall A/C arası, orb B — 3 site utility
- Skye: Flash kapı arkası, dog bilgi toplama çok iyi

## POST-PLANT STRATEJİLERİ
- A: A Main + Root crossfire, Tree'yi smoke'la
- B: B Main + kırık duvar crossfire, lineup çok etkili
- C: C Main + Mound crossfire, Waterfall'ı izle
- Kapı kullanarak post-plant pozisyon değiştir — defender'ı şaşırtır
- 3 site harita = retake süresi uzun, post-plant lineup'lar çok değerli

## EKONOMİ STRATEJİLERİ
- Attacker-sided harita — saldırı tarafı ekonomiyi agresif kullanabilir
- Force buy: C rush (en az utility ile yapılabilir)
- Eco: Rotating door ile agresif flank play (Spectre veya Sheriff)
- Yıkılabilir duvar kırmak için ek maliyet hesapla (ability charge)
- 3 site = utility çok önemli, tam buy'da ability eksik bırakma

## ANTI-STRAT İPUÇLARI
- Rakip her round B duvarı kırıyorsa: duvar arkasında bekle, kırıldığında trade al
- Rakip kapı rotasyonu yapıyorsa: kapı yakınında sentinel utility koy
- Rakip A Silent Drop kullanıyorsa: Drop altına tripwire veya turret
- Rakip C Mound agresifse: early smoke Mound + flash
- Rakip 3 site arasında yayılıyorsa: mid kontrolü al, split yap
