# HAVEN — Detaylı Strateji ve Analiz

## Harita Karakteristikleri
- 3 site (A, B, C) — oyundaki tek 3-site harita (Lotus ile birlikte)
- Savunma tarafı 5 kişiyle 3 site'ı tutmak zorunda — her zaman bir site zayıf
- Garage/Mid bölgesi A ve C arasında geçiş sağlar
- A Long ve C Long güçlü Op sightline'ları
- B site dar giriş, kolay savunulur
- Mid Window (Garage) bilgi ve rotasyon için kritik

## SALDIRI STRATEJİLERİ

### Default (1-3-1 veya 1-2-2)
- 1 kişi A Long (bilgi, Op bait)
- 2-3 kişi Mid/Garage (kontrolü al, B veya C'ye split imkanı)
- 1-2 kişi C Long (bilgi, hızlı execute hazırlığı)
- Garage kontrolü alınınca C veya A'ya split yapılır
- Defender rotasyonlarını izle, en zayıf site'a execute

### A Execute
- Smoke: Heaven + CT/Sewers çıkışı
- Flash: A Long'dan site'a doğru pop flash
- Entry: A Long → A Site, destek A Short/Sewers üzerinden
- A Short'tan (Sewers) split çok güçlü — rotate'u keser
- Spike: A Default (kutu arkası, Heaven'dan korunur)
- Post-plant: A Long + A Short crossfire, Heaven'ı izle

### B Execute
- Smoke: B Back Site (CT) + Window (Garage tarafı)
- B Garage'dan flash + push — dar giriş ama hızlı
- B Main giriş çok dar: utility dump zorunlu
- Entry: 2 kişi B Main, 1 kişi Mid Window'dan destek
- Spike: B Default (ortada)
- Post-plant: B Main + Mid Window crossfire
- B en zor execute: dar giriş, kolay molly/nade

### C Execute
- Smoke: C Long cross (Plat) + CT spawn
- Flash: C Long'dan site'a
- C Garage split: 2 kişi C Long, 2 kişi Garage → C
- Entry: C Long push + Garage desteği
- Spike: C Default (kutuların arkası)
- Post-plant: C Long + Garage crossfire

### Mid/Garage Control
- Garage kontrolü = haritanın anahtarı
- Mid Window'dan bilgi al, B'ye veya C'ye karar ver
- Garage'da Omen shrouded step veya Jett dash ile agresif bilgi
- Garage alınırsa B Window + C Connector açılır
- Garage smoke + flash combo ile entry yap

## SAVUNMA STRATEJİLERİ

### Default Savunma (2-1-2 veya 1-2-2)
- A: 1 A Heaven, 1 A Short/Sewers
- B: 1 kişi B Site (anchor) — en zor pozisyon
- C: 1 C Long, 1 C Plat/Connector
- Rotate: B anchor bilgi verirse A veya C'den 1 kişi gelir

### A Site Savunma
- A Heaven = en güçlü anchor ama sıkça flash yenir
- A Short/Sewers pozisyonu flank dinleme + retake
- A Long'da agresif Op oynayabilirsin ama trade desteği lazım
- Hell (A Site altı) off-angle clutch pozisyonu

### B Site Savunma
- B anchor = en stresli pozisyon, yalnız kalırsın
- B Window (Mid tarafı) ile Garage bilgisi al
- B Back Site'ta oyna, utility ile geciktir
- Killjoy/Cypher B'de çok güçlü — turret + tripwire
- Sentinel utility olmadan B tutmak çok zor

### C Site Savunma
- C Long'da 1 kişi — Op veya bilgi pozisyonu
- C Plat güçlü anchor, yükseklik avantajı var
- Connector ile B veya Garage'a hızlı rotate
- C Cubby'de off-angle oynama — 1 kill al geri çekil

### Garage/Mid Savunma
- Garage Window'dan bilgi al, gereksiz peek atma
- Garage kontrolünü kaybetme — kaybedersen 2 site tehlike altında
- Garage'da utility (Sage wall, KJ turret) ile delay
- Flank Garage üzerinden çok güçlü

## YAYGIN HATALAR (Rank Bazlı)

### Gold-Plat
- 3 site'ı 2-2-1 tutup Garage'ı boş bırakmak
- A Long'da solo Op oynayıp trade olmadan ölmek
- B anchor'ın yardım istemeden ölmesi (komünikasyon eksikliği)
- Her round aynı site'a execute (okunuyorsun)
- C Long'da 5-man push (rotate yenilir)

### Diamond-Ascendant
- Garage kontrolünü verip retake oynamamak
- B'ye over-rotate (A veya C boş kalır)
- Post-plant'te spike'a yakın durma (utility ile ölürsün)
- Fake'lere düşme (utility ses takibi zayıf)
- Eco round'da disiplin kaybı

### Immortal+
- Aynı default'u tekrar etme — varyasyon şart
- Mid-round karar gecikmesi (3 site = bilgi çok, karar hızlı olmalı)
- Timeout kullanmamak (momentum özellikle Haven'da çok önemli)
- Anti-eco'da spread oynamamak (Sheriff 1-tap riski)
- Post-plant lineup'larına karşı önlem almamak

## PRO TAKIMLAR — HAVEN META

### Fnatic Haven Stili
- Agresif Garage kontrolü (Boaster IGL agresif mid play)
- B'yi minimal utility ile tutma, A ve C'ye yoğunlaşma
- C Long agresif Op (Derke/Chronicle)
- Eco round: B rush Garage üzerinden

### LOUD Haven Stili
- Default-heavy, bilgi topla → mid-round commit
- aspas lurk C tarafında, takım A execute
- Pancada entry A Long → trade chain
- Saadhak Initiator utility ile bilgi toplama çok iyi

### DRX Haven Stili
- Methodical, yavaş execute
- 3 site arasında fake + rotate
- BuZz agresif Op C Long
- Post-plant lineup mastery (Viper + Brimstone)

## ÖNEMLİ POZİSYONLAR VE CALLOUT'LAR
- A Heaven / A Hell / A Short / A Long / A Sewers
- B Site / B Back / B Window / B Main / B Garage
- C Long / C Plat / C Cubby / C Connector / C Back
- Garage / Mid Window / Mid Doors
- CT Spawn (Defender Spawn) / T Spawn (Attacker Spawn)

## AGENT-SPECIFIC İPUÇLARI
- Omen: Garage shrouded step ile bilgi + B Heaven TP
- Killjoy: B anchor en iyi rolü — turret + alarmbot + lockdown
- Sova: Recon bolt A Long veya C Long başlangıcında bilgi
- Breach: B Main'den flash + aftershock combo
- Jett: C Long agresif Op, dash ile geri çekilme
- Astra: Stars ile 3 site'ı kontrol etme, Garage deny
- Cypher: B tripwire + cam, Garage bilgisi

## POST-PLANT STRATEJİLERİ
- A: A Long + A Short crossfire, Heaven'ı smoke'la
- B: B Main + Mid Window crossfire, B Back'i izle
- C: C Long + Garage crossfire, Plat'i smoke'la
- Haven'da post-plant lineup'lar çok değerli (3 site = rotate uzun)
- Spike'ı dinleme: defuse sesi duyunca peek, erken peek yapma

## EKONOMİ STRATEJİLERİ
- 3 site olduğu için utility çok önemli — tam buy'da utility eksik bırakma
- B anchor'a Sentinel verirsen utility maliyeti düşer
- Force buy: B rush (en az utility gerektiren execute)
- Eco: Garage üzerinden agresif play (dar koridor = Sheriff etkili)
- Bonus round: utility save et, gun advantage'ı kullan

## ANTI-STRAT İPUÇLARI
- Rakip her round A Long push yapıyorsa: Op + tripwire + agresif A Short flank
- Rakip Garage kontrolü alıyorsa: Sage wall + early aggression Window'dan
- Rakip B rush yapıyorsa: Killjoy lockdown + double stack B
- Rakip C Long Op oynuyorsa: smoke + 5 flash execute
- Rakip lurk-heavy oynuyorsa: flank watch utility (Cypher, KJ) zorunlu
