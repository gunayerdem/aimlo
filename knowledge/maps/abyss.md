# ABYSS — Detaylı Strateji ve Analiz

## Harita Karakteristikleri
- 2 site (A, B), haritanın kenarlarında uçurum (void/fall hazard)
- Duvar yok bazı bölgelerde — haritanın kenarından düşebilirsin
- Uçurum = unique rotation ve pozisyonlama fırsatları
- A site geniş, açık alan
- B site daha kapalı, nested yapı
- Mid bölgesi Bridge üzerinden bağlantı
- Yüksek risk-yüksek ödül pozisyonları (kenar oynama)

## SALDIRI STRATEJİLERİ

### Default (2-1-2)
- 2 kişi A Main/Ramp (bilgi, A kontrolü)
- 1 kişi Mid Bridge (bilgi, split imkanı)
- 2 kişi B Main/Lobby (bilgi al)
- Uçurum kenarlarını kullanarak unexpected angle'lar oluştur
- Mid Bridge kontrolü = her iki site'a split

### A Execute
- Smoke: A Heaven + A CT çıkışı
- Flash: A Main'den site'a doğru
- A Ramp kontrolü şart
- Uçurum tarafından geniş swing = unexpected angle
- Mid → A Link split çok güçlü
- Entry: A Main/Ramp push + Mid desteği
- Spike: A Default (güvenli pozisyon, uçurumdan uzak)
- Post-plant: A Main + A Link crossfire

### B Execute
- Smoke: B Site CT + B Heaven
- Flash: B Main/Lobby'den site'a
- B Lane kontrolü önemli
- Entry: B Main push + B Link desteği
- Uçurum kenarından peek = riskli ama etkili off-angle
- Spike: B Default (ortada)
- Post-plant: B Main + B Link crossfire

### Mid Bridge Control
- Bridge = Abyss'in merkezi
- Bridge'den A veya B'ye split imkanı
- Bridge'de pozisyon alırken uçuruma dikkat
- Bridge kontrolü rotate kararı için kritik
- Op Bridge'de etkili ama riskli (düşersen ölürsün)

### Void/Edge Plays
- Uçurum kenarlarında pozisyon al — düşman beklemez
- Jett/Raze ile uçurum üzerinden satchel/dash
- Kenar pozisyonları bilgi ve off-angle için çok güçlü
- Risk: düşme = ölüm, dikkatli ol
- Ability ile düşman düşürme: Breach aftershock, Sage slow kenar

## SAVUNMA STRATEJİLERİ

### Default Savunma (2-1-2)
- A: 1 A Heaven, 1 A Site/Ramp
- Mid: 1 kişi Bridge kontrolü
- B: 1 B Heaven/Site, 1 B Main izleme

### A Site Savunma
- A Heaven = güçlü anchor (yükseklik avantajı)
- Uçurum tarafına dikkat — düşmanlar kenardan gelebilir
- A Ramp'ta agresif bilgi al, geri çekil
- Off-angle uçurum kenarında çok etkili

### B Site Savunma
- B Heaven pozisyonu güçlü
- B Lane'de early aggression bilgi verir
- Uçurum kenarında Sage wall = unique delay
- B Back'te off-angle oynama

### Mid/Bridge Savunma
- Bridge kontrolü = her iki site güvenliği
- Bridge'de utility ile delay (Sage wall, smoke)
- Bridge'i kaybetme — kaybedersen split yenilir
- Dikkat: Bridge'de dururken uçuruma düşme riski

## YAYGIN HATALAR (Rank Bazlı)

### Gold-Plat
- Uçurumdan düşmek (haritayı öğren, kenarları bil)
- Kenar pozisyonlarını kullanmamak (haritanın unique avantajı)
- Bridge'i ignore etmek (mid kontrol çok önemli)
- Dar alanlarda 5 kişi sıkışmak
- Ability ile kendi takım arkadaşını uçurumdan düşürmek

### Diamond-Ascendant
- Heaven pozisyonlarında çok uzun durma
- Bridge kontrolünü verip retake oynamamak
- Post-plant'te riskli kenar pozisyon almak (düşme riski)
- Uçurum kenarında gereksiz duel almak
- Eco round'da disiplin kaybı

### Immortal+
- Kenar pozisyonlarını her round kullanma (okunur)
- Anti-strat adaptasyonu yapmamak
- Mid-round karar gecikmesi
- Timeout kullanmamak
- Uçurum mechaniğini anti-strat'a dahil etmemek

## PRO TAKIMLAR — ABYSS META

### Fnatic Abyss Stili
- Boaster IGL void plays (agresif kenar pozisyonları)
- Alfajer Jett satchel void üzerinden entry
- Leo Breach aftershock void kenarına (düşman düşürme)
- Derke Op Bridge dominant
- Unique A execute: kenar üzerinden split

### Sentinels Abyss Stili
- TenZ agresif Bridge Op
- zekken lurk kenar pozisyonları
- Killjoy B anchor (turret void kenarı izleme)
- Default-heavy, bilgi → commit

### Gen.G Abyss Stili
- t3xture Op multi-angle (Bridge + A Long)
- Lakia lurk void kenarlarında
- Methodical execute: utility dump → site entry
- Post-plant lineup'lar çok güçlü

## ÖNEMLİ POZİSYONLAR VE CALLOUT'LAR
- A Main / A Ramp / A Heaven / A Site / A Link / A Edge (Void tarafı)
- B Main / B Lobby / B Lane / B Heaven / B Site / B Link / B Edge
- Mid Bridge / Mid Top / Mid Bottom / Mid Connector
- CT Spawn / T Spawn / Void (uçurum bölgeleri)

## AGENT-SPECIFIC İPUÇLARI
- Jett: Void üzerinden dash entry, Bridge agresif Op
- Raze: Satchel void üzerinden site entry, boombot kenar kontrolü
- Sage: Wall Bridge = çok güçlü, wall kenar pozisyon oluşturma
- Breach: Aftershock void kenarına = düşman düşürme, fault line Bridge
- Viper: Wall mid split, orb site, void kenarına toxic screen
- Killjoy: Turret void kenarı izleme, alarmbot Bridge kontrolü
- Omen: TP void kenarından unexpected angle, shrouded step Bridge
- Fade: Prowler kenar kontrolü, haunt bilgi toplama

## POST-PLANT STRATEJİLERİ
- A: A Main + A Link crossfire, Heaven smoke'la
- B: B Main + B Link crossfire, Heaven'ı izle
- Void kenarında post-plant pozisyon = riskli ama çok güçlü off-angle
- Spike'ı uçurumdan uzak yere plant et (defuse eden düşmemesin = predictable pozisyon)
- Lineup'lar: Brimstone molly, Viper snake bite çok etkili

## EKONOMİ STRATEJİLERİ
- Op Bridge'de çok güçlü — Op economy önemli
- Force buy: B rush (daha kapalı alan, Spectre etkili)
- Eco: Bridge agresif Sheriff play, kenar pozisyondan pick
- Vandal > Phantom (açık alanlar, uzun sightline'lar var)
- Kenar pozisyonlarda shotgun (Judge, Bucky) eco'da etkili olabilir

## ANTI-STRAT İPUÇLARI
- Rakip her round Bridge push yapıyorsa: Sage wall Bridge + double utility
- Rakip void kenarlarını kullanıyorsa: Breach aftershock/fault line kenar
- Rakip A rush yapıyorsa: A Ramp'ta utility dump + Heaven crossfire
- Rakip B split yapıyorsa: B Lane'de sentinel utility + Bridge kontrolü
- Rakip agresif void plays yapıyorsa: kenarları molly/slow ile deny et
